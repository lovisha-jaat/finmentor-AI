💰 FinMentor AI – Personal Finance Mentor for India

FinMentor AI is an AI-powered personal finance web application designed to help Indian users plan, save, invest, and grow their money intelligently. It simplifies complex financial decisions into a user-friendly experience similar to chatting on WhatsApp.

🚀 Problem Statement

95% of Indians do not have a structured financial plan. Financial advisors are expensive and mostly accessible to high net-worth individuals. As a result, students and middle-class users struggle with:

Saving money effectively
Choosing the right investments
Understanding taxes
Planning for retirement
💡 Solution

FinMentor AI acts as a personal financial mentor, helping users:

Analyze their financial condition
Get a Money Health Score
Plan investments using SIP, PPF, NPS, and ELSS
Optimize taxes
Achieve financial independence (FIRE)

All in one simple platform.

🔥 Key Features
📊 Money Health Score
Calculates a score (0–100) based on savings, debt, investments, and retirement readiness
Provides actionable insights and improvement tips
🧓 FIRE Planner (Retirement Planning)
Predicts retirement age
Suggests monthly SIP investment
Displays wealth growth graph
💸 Tax Planner (India-Specific)
Suggests deductions under:
80C (ELSS, PPF)
80D (Insurance)
NPS (80CCD)
Compares Old vs New Tax Regime
🎯 Goal-Based Planning
Set goals (Car, House, Travel)
Calculates required monthly investment
Tracks progress visually
📈 What-If Simulator
Adjust income, savings, or expenses
Instantly see impact on:
Retirement age
Wealth growth
🧠 AI Financial Chat Assistant
WhatsApp-style interface
Provides personalized financial advice
Explains concepts in simple language
🏦 Net Worth Tracker
Tracks assets and liabilities
Calculates total net worth
📊 Expense Analytics
Visualizes spending patterns
Suggests ways to reduce expenses
🇮🇳 India-Focused Approach

This app is designed specifically for Indian users and supports:

SIP (Systematic Investment Plan)
PPF (Public Provident Fund)
NPS (National Pension System)
ELSS (Equity Linked Saving Scheme)
🛠️ Tech Stack
Frontend: React
Backend: Node.js / API routes
AI Integration: OpenAI API
Database: Firebase / MongoDB
Deployment: Vercel
🎯 Unique Value Proposition

Unlike traditional finance apps that only allow investing, FinMentor AI helps users:

Decide what to invest in
Understand why to invest
Plan how much to invest

All powered by AI.
